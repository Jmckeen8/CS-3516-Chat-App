This code has been tested to work in the following conditions:
-Windows 10 version 1909
-Java JDK 8 Update 221

The server and client are programmed to bind to localhost on port 4315
Windows firewall may ask for permission to allow Java to use the network, please allow

Both the client and server should be executed from the windows command prompt
Navigating to the directory with the appropriate .java file and running 
"java ClientMain" or "java ServerMain" will start that portion of the program
